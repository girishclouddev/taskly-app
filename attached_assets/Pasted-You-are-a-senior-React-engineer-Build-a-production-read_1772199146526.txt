--------------------------------------

2) Initialize Firebase
3) Setup Firebase Cloud Messaging

5) Request notification permission on first load
6) Generate and log FCM device token

REMINDER BEHAVIOR:

When task time matches:

A) Show in-app center modal
B) Send FCM push notification
C) Notification must:
    - Show title: "Task Reminder"
    - Show task name
    - Include icon
    - Auto open app when clicked
    - Focus browser tab

If user is:
- In different tab → show push notification
- Browser minimized → show push notification
- Tab closed but browser open → show push notification
- Browser reopened → service worker handles it

On clicking notification:
- Redirect to homepage
- Auto-open that task reminder modal

- Compare current date/time with tasks
- Trigger reminder once
- Prevent duplicate triggers
